# FitMe Backend ⚙️

> 대학생 맞춤형 장학금 · 공모전 매칭 서비스 — 백엔드

## 기술 스택

| 구분 | 기술 |
|------|------|
| Framework | Spring Boot |
| Language | Java |
| DB | MySQL |

## 브랜치 전략

- `main` : 최종 배포
- `develop` : 통합 개발
- `feat/backend` : 기능 개발 → develop으로 PR

## 시작하기

```bash
./gradlew bootRun
```
